# FPGA Mini-ISP Starter Block

A streaming 3×3 convolution accelerator in Verilog for the Microchip PolarFire SoC (MPFS025T / BeagleV-Fire).

## Repository Structure

- rtl/       : RTL modules
- tb/        : Testbenches
- scripts/   : Python scripts
- kernels/   : Convolution kernels
- docs/      : Documentation
- results/   : Outputs and reports
- libero/    : Libero project files
